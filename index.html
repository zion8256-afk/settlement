📖 Google 시트 연동 방법

1. sheets.new 접속 → 새 스프레드시트 열기

2. 상단 메뉴 → 확장 프로그램 → Apps Script 클릭

3. 기존 코드 전부 지우고 아래 버튼으로 붙여넣기

4. 저장(Ctrl+S) → 배포 → 새 배포

5. 유형: 웹 앱 → 액세스: 모든 사용자 → 배포

6. 생성된 URL을 위 입력칸에 붙여넣고 연결
